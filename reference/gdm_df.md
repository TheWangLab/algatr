# Create dataframe of GDM results

Create dataframe of GDM results

## Usage

``` r
gdm_df(gdm_result)
```

## Arguments

- gdm_result:

  output of
  [gdm_run](https://thewanglab.github.io/algatr/reference/gdm_run.md)

## Value

dataframe of gdm model coefficients
