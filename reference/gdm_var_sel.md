# Get best set of variables from a GDM model

Get best set of variables from a GDM model

## Usage

``` r
gdm_var_sel(gdmData, sig = 0.05, nperm = 10)
```

## Arguments

- gdmData:

  data formatted using GDM package

- sig:

  sig level for determining variable significance

- nperm:

  number of permutations to run for variable testing
