# Helper function for TESS barplots using ggplot

Helper function for TESS barplots using ggplot

## Usage

``` r
ggbarplot_helper(dat)
```

## Arguments

- dat:

  Q matrix

## Value

barplot with Q-values and individuals, colorized by K-value
