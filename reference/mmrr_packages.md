# Install MMRR packages

Checks for the presence of packages required for MMRR. If the package is
not already installed, it will automatically install it.

## Usage

``` r
mmrr_packages()
```

## Value

None

## Details

The following package will be installed if not already present:

- "GGally"

## Examples

``` r
if (FALSE) mmrr_packages() # \dontrun{}
```
