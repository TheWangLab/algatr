# Load example data

Example dataset from [Bouzid et al.
2022](https://doi.org/10.1111/mec.15836). The code used to create this
dataset can be found under data-raw/DATASET

## Usage

``` r
load_algatr_example(quiet = FALSE)
```

## Arguments

- quiet:

  whether to operate quietly and suppress messages (defaults to FALSE)

## Value

loads liz_vcf, liz_gendist, liz_coords, and CA_env objects
