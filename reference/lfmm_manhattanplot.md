# LFMM Manhattan Plot

LFMM Manhattan Plot

## Usage

``` r
lfmm_manhattanplot(df, sig, group = NULL, var = NULL)
```

## Arguments

- df:

  dataframe of lfmm test results produced by `lfmm_df`

- sig:

  significance cutoff

## Value

Manhattan plot
