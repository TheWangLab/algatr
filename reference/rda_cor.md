# Genotype-environment correlation test

Genotype-environment correlation test

## Usage

``` r
rda_cor(gen, var)
```

## Arguments

- gen:

  dosage matrix

- var:

  dataframe with predictor variables

## Value

dataframe with r and p-values from correlation test
