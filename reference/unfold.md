# unfold converts the lower diagonal elements of a matrix into a vector

unfold converts the lower diagonal elements of a matrix into a vector

## Usage

``` r
unfold(X, scale = TRUE)
```

## Arguments

- X:

  is a distance matrix

- scale:

  if TRUE then matrices will be standardized (defaults to TRUE)
