# Best K Selection based on cross entropy

Best K Selection based on cross entropy

## Usage

``` r
bestK(tess3_obj, Kvals)
```

## Arguments

- tess3_obj:

  list produced by `tess3`

- Kvals:

  vector of K values for testing
