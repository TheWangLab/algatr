# Install wingen packages

This function checks for the presence of packages required for wingen.
If the packages are not already installed, it will automatically install
them.

## Usage

``` r
wingen_packages()
```

## Value

None

## Details

The following package will be installed if not already present:

- "wingen"

## Examples

``` r
if (FALSE) wingen_packages() # \dontrun{}
```
