# Helper function to tidy LFMM test results

Helper function to tidy LFMM test results

## Usage

``` r
lfmm_test_tidy(colname, lfmm_test_result)
```

## Arguments

- colname:

  names of elements within lfmm_test_result

- lfmm_test_result:

  LFMM test results
