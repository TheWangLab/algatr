# Install GDM packages

Checks for the presence of packages required for GDM. If any of these
packages are not already installed, it will automatically install them.

## Usage

``` r
gdm_packages()
```

## Value

None

## Details

The following packages will be installed if not already present:

- "cowplot"

- "gdm"

## Examples

``` r
if (FALSE) gdm_packages() # \dontrun{}
```
